#### Sastantua

The goal of this program is to build the Sastantua pyramid using a given size as argument. Several sizes can be given in the same command line.

```bash
go run ./sastantua 1 3
```

**Output**

```bash
Size 1
  /*\
 /***\
/**|**\



Size 3
               /*\
              /***\
             /*****\
          /***********\
         /*************\
        /***************\
       /*****************\
    /***********************\
   /*************************\
  /************|||************\
 /*************|$|*************\
/**************|||**************\
```
